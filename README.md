# CCCFFM-Stairs
Wir (Dimi und ich) möchten im Space die Treppen zur Flounge verbessern.

## Modernisierungen:
- Beleuchtete Stufen mithilfe von Acrylglas
- Neues Laminat für die Treppenstufen
- Inbetriebnahme des leuchtenden Geländers
- Adressierbarkeit durch WLED (WT32-ETH01)
- Ethernetifizierung der Flounge
